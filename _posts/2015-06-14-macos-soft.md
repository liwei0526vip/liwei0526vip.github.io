## MacOS 必备软件清单

> * item2 + sshpass
> * vscode
> * 有道云笔记
> * chrome


## brew 安装

```bash
# 参考链接: https://brew.sh
$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
# 安装过程需要几分钟，需要耐心等待
```


## sshpass 安装

```bash
$ brew install https://raw.githubusercontent.com/kadwanev/bigboybrew/master/Library/Formula/sshpass.rb
```
